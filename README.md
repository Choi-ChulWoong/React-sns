## React SNS
https://reactsns.com
> React 기반의 SPA 페이지입니다.

> 기간 2019.10 ~ 2019.12  /  1인 프로젝트

> rest api를 사용한 crud 페이지를 구현하였습니다.




### Project Stack

#### Client

front-end에 사용한 기술 스택:

- React
- Redux
- Redux-Saga
- next.js (ssr)
- Immer
- Ant Design
- Styled-Components

#### Server

back-end에서 사용한 핵심 기술 스택:

- Node.js (express)
- Sequelize
- MySql
- AWS
